                                            DeepS Artyom Semyrozum • 8 months ago

                                            I think this was just an example that you can do it the '{% block heading %}' way as well.

                                            essentially, if you want to include a custom heading in your child template, you put that in {% block heading %} but instead if you do not want to put a custom heading, you just use super() to insert whatever was there in parent template.

                                            You could also do a mixture of both:

                                            {% block heading %}
                                            {{ super() }}

                                            Some custom heading apart from what's inserted by super()
                                            {% endblock %}
